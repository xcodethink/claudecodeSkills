# 11-数据可视化

| # | 技能 | 来源 | 难度 | 亮点 |
|---|------|------|------|------|
| 1 | [Matplotlib](Matplotlib.md) | dataviz | intermediate | Python 最基础的可视化库：静态、动画和交互式图表，出版级质量 |

| 4 | [出版级可视化](出版级可视化.md) | dataviz | advanced | 出版级科学可视化：多面板布局、误差线、显著性标记、色盲安全调色板 |
| 5 | [NetworkX](NetworkX.md) | dataviz | advanced | 复杂网络与图的分析可视化：社交网络、知识图谱、生物网络 |
