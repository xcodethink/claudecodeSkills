# 图片增强

此技能获取你的图片和截图并使其看起来更好 — 更锐利、更清晰、更专业。提升图片质量，特别是截图的分辨率、锐度和清晰度。非常适合为演示文稿、文档或社交媒体帖子准备图片。

## 何时使用此技能

- 提升博客文章或文档中的截图质量
- 在社交媒体分享前增强图片
- 为演示文稿或报告准备图片
- 放大低分辨率图片
- 锐化模糊的照片
- 清理压缩产生的伪影

## 此技能的功能

1. **分析图片质量**：检查分辨率、锐度和压缩伪影（Compression Artifacts）
2. **增强分辨率**：智能放大图片
3. **提升锐度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪点
5. **按用途优化**：根据预期用途调整（网页、打印、社交媒体）

## 使用技巧

- 始终保留原始文件作为备份
- 对截图和数字图片效果最佳
- 可批量处理整个文件夹
- 如需要可指定输出格式（PNG 保质量、JPG 缩体积）
- 针对社交媒体，请说明平台以获取最佳尺寸

## 常见用例

- **博客文章**：发布前增强截图
- **文档**：使 UI 截图清晰锐利
- **社交媒体**：为 Twitter、LinkedIn、Instagram 优化图片
- **演示文稿**：为大屏幕放大图片
- **打印材料**：为实体印刷提升分辨率
