# 域名头脑风暴器（Domain Name Brainstormer）

本技能通过生成创意域名选项并检查哪些域名实际可注册，帮助你为项目找到完美的域名。

## 适用场景

- 启动新项目或公司
- 发布产品或服务
- 创建个人品牌或作品集网站
- 为现有项目重新品牌化
- 为副业项目注册域名
- 当首选域名已被注册时寻找可用替代方案

## 功能说明

1. **理解项目**：分析你正在构建什么以及目标用户是谁
2. **生成创意名称**：创建相关的、易记的域名选项
3. **检查可用性**：验证哪些域名在多个 TLD（顶级域名）中实际可用
4. **多种后缀**：建议 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果首选被占用，提供变体
6. **品牌洞察**：解释为什么某些名称效果好

## 使用方法

### 基础头脑风暴

```
I'm building a project management tool for remote teams.
Suggest domain names.
（我正在构建一个面向远程团队的项目管理工具。建议域名。）

```

## 域名命名技巧

### 好域名的特征

- **简短**：理想情况下 15 个字符以内
- **易记**：容易回忆和拼写
- **可发音**：可以在对话中说出来
- **描述性**：暗示你做什么
- **可品牌化**：足够独特，能脱颖而出
- **无连字符**：更容易口头分享

### TLD（顶级域名）指南

- **.com**：通用、可信，适合各类商业
- **.io**：科技创业公司、开发者工具
- **.dev**：面向开发者的产品
- **.ai**：AI/ML（人工智能/机器学习）产品
- **.app**：移动或 Web 应用
- **.co**：.com 的替代选择
- **.xyz**：现代、创意项目
- **.design**：创意/设计机构
- **.tech**：科技公司

## 示例工作流

### 创业公司发布
1. 描述你的创业想法
2. 获取 10-15 个跨 TLD 的域名建议
3. 查看可用性和定价
4. 选出前 3 个最喜欢的
5. 立即注册

### 个人品牌
1. 分享你的名字和职业
2. 获取变体（firstname.com、firstnamelastname.dev 等）
3. 同时检查社交媒体用户名可用性
4. 在各平台注册一致的品牌

### 产品命名
1. 描述产品和目标市场
2. 获取创意、可品牌化的名称
3. 检查商标冲突
4. 验证域名和社交媒体可用性
5. 在目标受众中测试名称

## 成功秘诀

1. **快速行动**：好域名很快就会被注册
2. **注册多个变体**：同时拿下 .com 和 .io 以保护品牌
3. **避免数字**：口头沟通时难以传达
4. **检查社交媒体**：确保 @username 也是可用的
5. **大声说出来**：测试是否容易发音
6. **检查商标**：确保没有法律冲突
7. **长远考虑**：5 年后这个名称还有意义吗？

## 定价参考

建议域名时会注明：
- 标准域名：约 $10-15/年
- 高级 TLD（.io、.ai）：约 $30-50/年
- 已注册域名：如有列出则按市场价
- 优质域名：数百到数千美元
