# 技能模板

这是一个技能（Skill）的基础模板。创建新技能时，以此为起点。

## SKILL.md 模板结构

```yaml
---
name: template-skill
description: 替换为技能的描述以及 Claude 应在何时使用它。
---

# 在下方插入指令
```

## 说明

- `name` 字段：技能的唯一标识名称，使用小写字母和连字符
- `description` 字段：描述技能的功能和触发条件，这决定了 Claude 何时会使用该技能
- YAML 前置元数据（frontmatter）之后是 Markdown 格式的指令正文

要使用此模板，将 `name` 和 `description` 替换为实际内容，然后在 YAML 前置元数据下方编写具体的技能指令。
