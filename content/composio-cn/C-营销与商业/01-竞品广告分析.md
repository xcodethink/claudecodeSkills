# 竞品广告分析器（Competitive Ads Extractor）

本技能从广告资料库中提取竞争对手的广告，分析其有效策略——包括他们强调的痛点、针对的使用场景，以及引起共鸣的文案/创意素材。

## 适用场景

- 研究竞争对手的广告策略
- 为自己的广告寻找灵感
- 了解市场定位
- 识别成功的广告模式
- 分析有效的营销信息
- 发现新的使用场景或用户痛点
- 基于已验证的概念规划广告活动

## 功能说明

1. **提取广告**：从 Facebook Ad Library（Facebook 广告库）、LinkedIn 等平台抓取广告
2. **截图保存**：保存所有广告的可视化副本
3. **信息分析**：识别痛点、使用场景和价值主张（Value Proposition）
4. **广告分类**：按主题、受众或格式进行分组
5. **模式识别**：发现常见的成功方法
6. **洞察输出**：解释某些广告表现良好的可能原因

## 使用方法

### 基础提取

```
Extract all current ads from [竞争对手名称] on Facebook Ad Library
```

```
Scrape ads from [公司名称] and analyze their messaging
```

### 特定分析

```
Get all ads from [竞争对手] focusing on their messaging
about [具体问题]. What pain points are they highlighting?
```

### 竞品集合分析

```
Extract ads from these 5 competitors: [列表].
Compare their approaches and tell me what's working.
```

### 指定平台

```
Get LinkedIn ads from [竞争对手] and analyze their
B2B positioning strategy
```

## 示例

**用户**："提取 Notion 在 Facebook Ad Library 上的广告，告诉我哪些营销信息是有效的。"

**处理过程**：
```
访问 Facebook Ad Library...
搜索：Notion
发现：23 条活跃广告

正在提取截图...
[████████████████████] 100%

已保存至：competitor-ads/notion/
- ad-001-collaboration.png
- ad-002-productivity.png
- ad-003-templates.png
...

正在分析营销信息...
```

**输出结果**：
```
# Notion 广告分析

## 概览
- 广告总数：23 条活跃广告
- 主要主题：生产力（35%）、协作（30%）、模板（20%）、AI 功能（15%）
- 广告格式：静态图片（60%）、视频（40%）
- CTA（行动号召）模式："Try for free（免费试用）"、"Get started（立即开始）"

## 他们强调的核心痛点

1. **信息分散**（8 条广告）
   文案："Stop switching between 10 different tools（别再在 10 个不同工具之间切换了）"
   有效原因：直击许多团队面临的痛点

2. **会议过载**（5 条广告）
   文案："Replace unnecessary meetings with async updates（用异步更新取代不必要的会议）"
   有效原因：后疫情时代远程办公的痛点

3. **文档丢失**（4 条广告）
   文案："Never ask 'where is that doc?' again（再也不用问'那个文档在哪'）"
   有效原因：职场中普遍存在的挫败感

## 成功的创意模式

### 模式 1：前后对比
- 展示混乱的工具场景 → 整洁的 Notion 工作空间
- 在 6 条高效广告中使用
- 视觉隐喻一目了然

### 模式 2：功能展示
- 实际产品使用的 GIF 动图
- 5 秒内展示特定功能
- 用于新功能推广（AI、模板）

### 模式 3：社会证明（Social Proof）
- "Join 20M users（加入 2000 万用户）" 的信息传递
- 客户 Logo 展示
- 在 4 条面向企业的广告中使用

## 有效文案分析

最佳标题：
1. "Your team's knowledge, finally in one place（你团队的知识，终于集中在一处）"
   → 聚焦利益，直接解决痛点

2. "The all-in-one workspace（一站式工作空间）"
   → 清晰定位，广泛吸引力

3. "AI that actually helps you work（真正帮你工作的 AI）"
   → 回应 AI 质疑，务实角度

最佳正文文案模式：
- 短句（10 个单词以内）
- 聚焦结果而非功能
- 包含具体数字（"Cut meetings by 50%（减少 50% 的会议）"）

## 受众定向洞察

基于广告变体分析：
- 创业者：个人生产力角度
- 团队负责人：协作与对齐
- 企业用户：安全性与合规性
- 学生：免费计划、模板、组织管理

## 对你广告的建议

1. **测试"工具分散"痛点**
   → 根据他们的广告投放频率，此痛点共鸣强烈

2. **使用产品截图而非抽象视觉素材**
   → 他们所有表现最好的广告都展示了实际 UI

3. **以问题开头，而非解决方案**
   → "Tired of X?（厌倦了 X？）" 的表现优于 "Introducing Y（介绍 Y）"

4. **文案保持在 100 字符以内**
   → 他们最短的广告似乎投放最频繁

5. **测试前后对比的视觉格式**
   → 在他们的创意素材中已验证的模式

## 已保存文件
- 所有广告：~/competitor-ads/notion/
- 分析报告：~/competitor-ads/notion/analysis.md
- 最佳表现：~/competitor-ads/notion/top-10/
```

## 你可以学到什么

### 信息分析
- 他们强调什么问题
- 如何针对竞争对手定位
- 引起共鸣的价值主张
- 目标受众细分

### 创意模式
- 有效的视觉风格
- 视频 vs 静态图片的表现
- 配色方案和品牌调性
- 布局模式

### 文案公式
- 标题结构
- 行动号召（CTA）模式
- 长度和语调
- 情感触发点

### 投放策略
- 季节性投放活动
- 产品发布方法
- 功能公告策略
- 再营销（Retargeting）模式

## 最佳实践

### 法律与道德
- 仅用于研究和灵感获取
- 不要直接复制广告
- 尊重知识产权
- 用洞察来指导原创创意
- 不要抄袭文案或窃取设计

### 分析技巧
1. **寻找模式**：哪些主题反复出现？
2. **长期追踪**：每月保存广告以观察演变
3. **验证假设**：为你的品牌调整成功模式
4. **按受众细分**：不同目标对应不同信息
5. **跨平台比较**：LinkedIn vs Facebook 的信息传递方式不同

## 常见工作流

### 广告活动策划
1. 提取竞品广告
2. 识别成功模式
3. 注意他们信息传递中的空白
4. 头脑风暴独特角度
5. 起草测试广告变体

### 定位研究
1. 获取 5 个竞品的广告
2. 绘制他们的定位图
3. 发现未被充分服务的角度
4. 开发差异化信息
5. 对比他们的方法进行测试

### 创意灵感
1. 按主题提取广告
2. 分析视觉模式
3. 记录配色和布局趋势
4. 调整成功模式
5. 创建原创变体

## 成功秘诀

1. **定期监控**：每月检查变化
2. **广泛研究**：也关注相邻领域的竞争对手
3. **保存一切**：建立参考素材库
4. **验证洞察**：进行自己的实验
5. **追踪效果**：对灵感概念进行 A/B 测试
6. **保持原创**：用于灵感而非抄袭
7. **多平台覆盖**：比较 Facebook、LinkedIn、TikTok 等

## 输出格式

- **截图**：所有广告保存为图片
- **分析报告**：Markdown 格式的洞察摘要
- **电子表格**：包含广告文案、CTA、主题的 CSV 文件
- **演示文稿**：最佳表现广告的视觉幻灯片
- **模式库**：按方法分类
