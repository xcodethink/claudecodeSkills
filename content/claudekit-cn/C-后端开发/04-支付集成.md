# 支付集成技能

经过生产验证的支付处理方案，涵盖 SePay（越南银行）、Polar（全球 SaaS）、Stripe（全球基础设施）、Paddle（MoR 订阅）和 Creem.io（MoR + 授权许可）。

## 适用场景

- 支付网关集成（结账、处理）
- 订阅管理（试用、升级、计费）
- Webhook 处理（通知、幂等性）
- 二维码支付（VietQR、NAPAS）
- 软件授权许可（License，设备激活）
- 多供应商订单管理
- 收入分成与佣金

## 平台选型

| 平台 | 最适合的场景 |
|------|-------------|
| **SePay** | 越南市场、越南盾（VND）、银行转账、VietQR |
| **Polar** | 全球 SaaS、订阅、自动化权益交付（GitHub/Discord） |
| **Stripe** | 企业级支付、Connect 平台、自定义结账 |
| **Paddle** | MoR（商户记录）订阅、全球税务合规、流失预防 |
| **Creem.io** | MoR + 授权许可、收入分成、无代码结账 |

## 核心能力

| 平台 | 核心亮点 |
|------|---------|
| **SePay** | 二维码/银行/银行卡支付，支持 44+ 家越南银行，Webhook，2 请求/秒 |
| **Polar** | MoR，订阅，用量计费，权益交付，300 请求/分钟 |
| **Stripe** | CheckoutSession，Billing 计费，Connect 平台，Payment Element |
| **Paddle** | MoR，弹出层/内嵌结账，Retain（流失预防），税务处理 |
| **Creem.io** | MoR，授权许可（Licensing），收入分成，无代码结账 |
