## 概述

从 K-Dense-AI 科学技能集中精选的 5 个数据可视化技能，适用于通用开发场景。涵盖静态图表（Matplotlib）、交互式可视化（Plotly）、统计图表（Seaborn）、出版级图表和网络图分析（NetworkX）。

## 目录

| 文件 | 核心内容 |
|------|----------|
| [01-Matplotlib](01-Matplotlib.md) | Figure/Axes 层级结构、pyplot vs 面向对象接口、图表类型、样式、3D 绘图 |
| [02-Plotly](02-Plotly.md) | Plotly Express vs Graph Objects、40+ 图表类型、交互特性、Dash 集成 |
| [03-Seaborn](03-Seaborn.md) | 统计可视化函数、关系/分布/分类/回归/矩阵图、调色板与主题 |
| [04-出版级可视化](04-出版级可视化.md) | 期刊规格（Nature/Science/Cell）、Okabe-Ito 色盲友好配色、多面板布局、投稿清单 |
| [05-NetworkX](05-NetworkX.md) | 图类型、中心性/最短路径/社区检测算法、布局算法、出版级网络图 |
